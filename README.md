# CycleCompanion

**CycleCompanion: A Machine Learning Approach to Personalized Women's Health, Wellness, and Workplace Productivity**

## Project Overview
This project focuses on creating a comprehensive web-based platform that uses machine learning to deliver personalized health and wellness guidance throughout different menstrual cycle phases. 

The platform helps women understand and optimize their daily routines—from exercise and nutrition to rest and workplace performance—by working with their body's natural hormonal patterns rather than against them.
